### What is Pawnyable?
Pawnyable is an intermediate-to-advanced learning site for Binary Exploitation and pwn.
It was originally written in Japanese because there were not many places to study advanced pwn topics in Japanese.

### Intended readers
This site assumes you are already comfortable with assembly, common memory corruption bugs such as BOF and UAF, and basic ROP construction.
