### Pawnyable이란?
Pawnyable은 Binary Exploitation, 즉 pwn을 공부하기 위한 중상급자용 학습 사이트입니다.
고급 pwn 주제를 일본어로 정리한 자료가 많지 않아서 제작된 사이트입니다.

### 대상 독자
어셈블리 읽기와 쓰기, BOF와 UAF 같은 기본 취약점 개념, ROP 체인 구성 정도는 이미 알고 있다는 전제로 작성되어 있습니다.

### 업데이트 계획
현재는 Linux Kernel 장 일부가 완성되어 있으며, 이후에는 다음 작업이 예정되어 있습니다.

- 우선순위 높음
  - Linux Kernel 장 완성
- 우선순위 중간
  - 예제 정답 추가
  - Practice 페이지 추가
  - VM Escape, Browser Exploit, Windows Kernel 장 추가
- 우선순위 낮음
  - Windows Userland, Linux Userland 장 추가
  - 다국어 지원 확장

문서는 [GitHub](https://github.com/ptr-yudai/pawnyable)에서 오픈소스로 공개되어 있으며, 오탈자 수정부터 새 장 추가까지 PR을 받고 있습니다.
