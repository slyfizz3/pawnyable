NX가 셸코드를 끝냈을까?

NX는 고전적인 주입 코드 실행을 어렵게 만들었지만, JIT 환경, sandbox escape, staged payload, 제한된 CTF 문제 등에서는 여전히 셸코드가 중요합니다.

JIT와 셸코드
