What is Pawnyable?

Pawnyable is an intermediate-to-advanced learning site for Binary Exploitation and pwn.
It was originally written in Japanese because there were not many places to study advanced pwn topics in Japanese.

Intended readers

This site assumes you are already comfortable with assembly, common memory corruption bugs such as BOF and UAF, and basic ROP construction.

Updates

Part of the Linux Kernel chapter is complete. Future work includes:

  • High priority
    • Completing the Linux Kernel chapter
  • Medium priority
    • Adding more answer writeups
    • Adding more practice pages
    • Adding VM Escape, Browser Exploit, and Windows Kernel sections
  • Low priority
    • Adding Windows Userland and Linux Userland content

The articles are open source on GitHub, and pull requests are welcome for edits, fixes, and new chapters.
